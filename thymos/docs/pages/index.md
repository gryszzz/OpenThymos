# Thymos

Thymos is the governed execution substrate for code agents.

Core thesis:

- cognition proposes
- runtime decides
- ledger remembers

This Pages site is the public surface for the runtime architecture and the
secure tool fabric now landing in Slice 2.

## What exists now

- signed writs and budgeted tool access
- append-only trajectory ledger
- runtime policy gates and approvals
- Anthropic, OpenAI, local OpenAI-compatible, and mock cognition
- secure tool fabric seam with worker-backed `shell` and `http`
