# Runtime Roadmap

The monster-product direction is:

**bring your own model, keep execution under Thymos**

## Build sequence

1. Lock the kernel
2. Production ledger and persistent state
3. Secure tool fabric
4. Model routing across premium, cheap, and local models
5. Governance and control plane
6. Trusted marketplace and distribution

## Current milestone

Slice 2 has started:

- worker seam added
- subprocess worker binary added
- hardened secure shell policy added
- HTTP private-host blocking added

The local repo contains the fuller roadmap in `docs/monster-runtime-roadmap.md`.
